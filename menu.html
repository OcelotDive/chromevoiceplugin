Open Tab : Opens new tab.

Goto Tab [number] :Navigates to tab number.

Close Tab / Exit Tab : Closes current tab.

Close Tab [number] : Closes the specified tab by position number.

Remove All Left : Closes all tabs to the left of the active extension menu.

Remove All Right : Closes all tabs to the right of the active extension menu.

Scroll [direction] : Scrolls in the specified direction (up/down/left/right/top/bottom).

Refresh : Refreshes current tab.

Search : Highlights searchbar and activates search mode.

When search mode is active the next spoken command will be searched for.

Elements : Sets a number selector next to each link on the page

When elements are displayed the next number spoken will navigate to chosen selection.

Cancel : Returns to default from search or elements mode.

Play : Plays a previously paused video.

Pause : Pause current video.

Forward [time]: Fast forwards by the selected time.

[time] can equal "thirty seconds", "one minute", "five minutes", "ten minutes", "thirty minutes".

Rewind [time]: Rewinds by the selected time.

[time] can equal "thirty seconds", "one minute", "five minutes", "ten minutes", "thirty minutes".

Restart : Restarts video playback from beginning.

Mute : Mutes video sound.

Sound : Un-mutes video sound.

Volume Up : Increases volume by incremental amount.

Volume Down : Decreases volume by incremental amount.

Captions : Turns on subtitles.

Cinema Mode : Enters youtube cinema mode.

Skip : Skip advert playing within video.

Remove ad : Removes ad popups within video.
